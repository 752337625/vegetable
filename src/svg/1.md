# SVG 动画如此简单

## SVG 基础动画

**默认动画中心是(0，0)点**

### transform="scale(x, y)"

::: normal-demo transform="scale(x, y)"

```html
<svg width="500" height="300">
  <rect x="0" y="0" width="100" height="100" fill="#ff770f" />
  <rect
    x="0"
    y="200"
    width="100"
    height="100"
    fill="#7e9178"
    transform="scale(0.5)"
  />
  <rect
    x="0"
    y="100"
    width="100"
    height="100"
    fill="#183c78"
    transform="scale(1.5)"
  />
  <rect
    x="200"
    y="100"
    width="100"
    height="100"
    fill="#cccccc"
    transform="scale(1, 0.5)"
  />
</svg>
```

:::

## JS 动画

### 控制整个 SVG 形成动画

::: normal-demo 控制整个 SVG 形成动画

```html
<svg width="500" height="300">
  <rect id="svg" x="0" y="0" width="100" height="100" fill="#ff770f" />
</svg>
```

```js
const svgEl = document.getElementById("svg");
let x = 0,
  y = 0,
  add = true;
setInterval(() => {
  x += 1;
  if (y < 150 && add) {
    y += 1;
  } else {
    add = false;
    y -= 1;
  }
  svgEl.setAttribute("transform", `translate(${x}, ${y})`);
}, 10);
```

:::

### 控制 SVG 内部元素形成动画

::: normal-demo 控制 SVG 内部元素形成动画

```html
<svg width="500" height="500" fill="orange">
  <path id="svgPath" />
</svg>
```

```js
const svgPath = document.getElementById("svgPath");
let x = 250,
  x1 = 250,
  y = 100;
setInterval(() => {
  if (x < 350) {
    x += 0.1;
  } else {
    x1 = 300;
    if (y < 200) y += 0.1;
  }
  svgPath.setAttribute("d", `M 250 100 300 0 ${x} 100 ${x1} ${y} z`);
}, 0);
```

:::
