# Socket.io 服务端

<!-- more -->

注意俩个对象

- ioHttps：全家
- socket：局部

```js:no-line-numbers
ioHttps.on('connection', function (socket) {
  ioHttps.emit('eventName',data) //向所有房间和人广播事件
  socket.emit('eventName',data) //向自己广播事件
  socket.broadcast.emit('eventName', data); //向除了自己以外的人广播事件
});
```
